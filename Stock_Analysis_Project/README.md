# Stock Analysis Dashboard

This project aims to provide retail investors with an intuitive tool for analysing technology stock trends.

## Features
- Interactive visualisation of price trends with 20-day moving average (MA20)
- Automatic generation of descriptive statistics (mean, volatility, and daily returns)
- User-friendly dashboard built with Streamlit

## How to Run
1. Install dependencies:
```bash
pip install -r requirements.txt
```
2. Launch the application:
streamlit run app.py
```
## Notes

This dashboard uses simulated stock data based on real price ranges, created for academic demonstration purposes only. It does not constitute financial advice.
